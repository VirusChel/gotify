# 🎧 GoTify — музыкальное сообщество

Лёгкий веб-плеер с загрузкой треков, плейлистами, темами и адаптивом.

## ✨ Возможности
- Загрузка аудио (mp3, wav, ogg)
- Общая лента треков сообщества
- Поиск и топ популярного
- Моя библиотека + плейлисты
- 5 тем оформления
- Полностью адаптивный дизайн
- Данные хранятся в браузере
